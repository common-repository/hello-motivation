A little plugin that will motivate you while your working in wordpress.

== Description ==

You know you need to blog, sometimes it can be a challenge and sometimes it is hard not to focus on the long term goal This is where Hello Motivtion will help, displaying random motivational quotes on the top right hand side of your wordpress screen while you are working in your blog.  This plugin is based on the Hello_Dolly plugin by Matt Mullenweg 

== Installation ==

1. Download and unzip the current version of the hello_motivation plugin.
2. Transfer the entire 'hello_motivation' directory to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. That's it! You're done.
